## Jest

#### Jest 폴더 구조
- 단위 테스트 폴더 -> \<filename>.test.js
- 통합 테스트 -> \<filename>.test.init.js

---

#### Jest가 test파일을 찾는 방법
- \<filename>.`test`.js
- \<filename>.`spect`.js
- `tests` folder

#### Jest 파일 구조
- describe: 그룹화하는 블록
- it(test): 개별 테스트를 수행

---

#### expect <-> matcher
- expect: 함수값
- matcher: 여러 방법으로 값을 테스트
